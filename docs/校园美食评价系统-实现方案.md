## ⚠️ 遇到问题怎么办

| 问题类型 | 最佳做法 |
|---------|---------|
| UI 不对、布局错乱 | **截图** + "我期望X，实际Y" |
| 点了没反应、报红字 | **截图报错信息** + "在哪个页面做了什么操作" |
| 不知道怎么描述 | 用大白话 + 截图标红框 |
| 想加 Skill 或功能 | 直接说"我想加一个 XX 的 skill / 功能" |
